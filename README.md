# 🏠 **House Rent App**  
Browse and explore rental houses easily with a simple Android app  

---

## 🚀 **Overview**  

**House Rent** is a lightweight Android application that helps users find and explore rental houses quickly and efficiently.  

The app is designed for **simplicity** — no complicated steps, just open the app, browse listings, view details, and find your desired house instantly.

## ✨ **Features**  

### 🏠 **House Listings**  
- Browse available rental houses  
- View house details (**price, location, rooms, etc.**)  
- Organized and clean listing system  

### 🔍 **Search & Filter**  
- Search houses based on preferences  
- Filter listings easily  
- Quick navigation between results  

### 💬 **Simple Interaction**  
- **User-friendly interface**  
- **Clean and minimal design**  
- Easy navigation for all users  

### ⚡ **Fast Performance**  
- Lightweight and efficient  
- Smooth and responsive UI  

---

## 📥 **How It Works**  
1. Open the app  
2. Browse available houses  
3. Use **search/filter** to find preferred houses  
4. Tap on any house to view details
